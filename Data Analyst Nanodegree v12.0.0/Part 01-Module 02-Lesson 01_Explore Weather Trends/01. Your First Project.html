Preparing for your project

This project requires the following steps:

  • Extract data from a database using a SQL query
  • Calculate a moving average in a spreadsheet
  • Create a line chart in a spreadsheet

If you know how to create SQL queries, you can proceed to the project instructions. Otherwise you might want to brush up on your basic SQL skills in the extracurricular section of your classroom first.
